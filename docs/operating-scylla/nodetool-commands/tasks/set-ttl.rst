Nodetool tasks set-ttl
======================
**tasks set-ttl** - Sets task_ttl value in seconds, i.e. time for which tasks are kept in task manager after
they are finished.

Syntax
-------
.. code-block:: console

   nodetool tasks set-ttl <time_in_seconds>

For example:

.. code-block:: shell

   > nodetool tasks set-ttl 10
